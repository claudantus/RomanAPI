# RomanAPI
REST API returning roman numerals in a specified range

# Useful commands
Run these in the ./src folder

## Initialize Module
```go mod init $modulename```

## Run Test
```go test```

## Get Dependency on Gin (for quick RestAPI)
```go get -u github.com/gin-gonic/gin```
